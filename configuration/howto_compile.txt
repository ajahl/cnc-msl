1) ROS install wiki: http://www.ros.org/wiki/ROS/Installation
(mirror
http://ros.informatik.uni-freiburg.de/roswiki/ROS%282f%29Installation.html)

2) follow the first 4 steps, (or 2.1 -2.6)

these are gentoo steps, for other distributions please use the wiki
2.0.1) set python useflag for subversion
2.1) install: pyyaml pysvn cmake subversion mercurial gentoolkit setuptools layman sudo
2.1.1) edit /etc/sudoers to allow user to become root via sudo for later
package installations
2.2) create ~/.pydistutils.cfg with content (for local installation of ROS)
[install]
user=1

2.3) change PATH: export PATH=$PATH:$HOME/.local/bin
2.4) easy_install -U rosinstall
2.4.1) fix $HOME/ros/rosdep.yaml (change dev-util/gtest to dev-cpp/gtest)
2.5) ROS Desktop-Full Install
rosinstall ~/ros "http://packages.ros.org/cgi-bin/gen_rosinstall.py?rosdistro=electric&variant=desktop-full&overlay=no"
2.6) change .bashrc (for local ROS install instead of /opt): source ~/ros/setup.bash

3) checkout our source: git clone ssh://impera@lanzarote.vs.eecs.uni-kassel.de/~/impera.git robocup

4) adapt bashrc according to configuration/bashrc

5) rosmake MSLCN Alica --threads=n
no more threads than CPU cores and not more than RAM/2Gb
